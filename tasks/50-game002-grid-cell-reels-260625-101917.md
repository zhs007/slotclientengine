## 3. 关键实现说明

- grid-cell order 是稳定的列优先顺序：`(0,0),(0,1)...(0,8),(1,0)...(5,8)`。
- 每个 cell 是一个独立 `RenderReel`，但所有通用算法在 rendercore 内部实现，game002 不直接操作 runtime 私有 Pixi container。
- 暗度不是固定盖在棋盘外层，而是每个微型 reel 内部的一条交替深浅 strip：
  - strip 随 `RenderReel.getSnapshot().currentY` 同步位移。
  - 可见行根据 reel 当前整数 y 与 cell order index 计算 `0.50 / 0.35`。
  - 单测会采样同一 cell 的 `dimmingAlpha`，要求出现 `0.50` 和 `0.35` 两种值。
- landed 后调用 `resetToVisibleSymbols(targetVisibleSymbols, finalY)` 并 reset 当前 slot symbol，避免 `RenderReel.land()` 的通用 `appear` 语义残留到 grid-cell 游戏表现。
- 最终 scene 验收通过 `getVisibleScene()` 与目标 scene 完全一致。
